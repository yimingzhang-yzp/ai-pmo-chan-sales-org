## 1. 日本市場 — 最も検索・利用されている TOP5

### 総評
日本市場では「国産ツール」と「グローバルツール日本語対応版」が混在して競争している。
2025〜2026年にかけてAI機能の追加が急速に進み、各社が差別化を図っている。
市場シェア調査（BOXIL調査）: Cloudlog 13.21% / Asana 7.07% / Backlog 6.47% が上位。

### 🇯🇵 #1 Backlog（バックログ）
**運営:** 株式会社ヌーラボ（福岡）  
**カテゴリ:** 国産プロジェクト管理ツール  

#### 概要
日本企業向けNo.1の国産プロジェクト管理ツール。IT・開発チームから一般業務チームまで幅広く利用。
2026年にAI機能を本格化。

#### AI機能（2026年現在）
| 機能 | 内容 |
|------|------|
| Backlog AIアシスタント | チャット形式でプロジェクト進捗レポートを自動生成 |
| AIバックログスイーパー | 会議内容からタスクを自動抽出・起票（アドオン） |
| AI課題一括登録 | プロジェクト計画をAIで自動作成 |
| AI要約機能 | 課題・コメントの要約 |

#### 価格（2026年）
| プラン | 料金 |
|--------|------|
| フリー | 無料（10名・1プロジェクト） |
| スターター | ¥2,970/月（30名まで） |
| スタンダード | ¥17,600/月 |
| プレミアム | ¥29,700/月 |

#### 強み
- 日本語完全対応・日本語ドキュメント充実
- 20年以上の国内実績・信頼性
- ガントチャート・課題管理が直感的
- 日本のサポート体制（ヌーラボ直接サポート）

#### 弱み
- AI機能は2026年に追加開始でまだ発展途上
- グローバル展開は限定的
- 高度なPMO自動化機能は競合より弱い

## 4. 自社PMO-AIの差別化ポイント（仮説）

1. **PMO業務に完全特化**  
   競合はプロジェクト管理ツールにAIを後付けしているが、PMO-AIはAIファーストで設計。
   計画策定・リスク管理・ステークホルダーレポートを自動化する点が根本的に異なる。

2. **日本語・日本企業文化への最適化**  
   会議文化（稟議・報告書）や日本固有の業務フローに対応。
   Backlogの「国産ツール安心感」+ClickUpの「AI先進性」を両立できる可能性。

3. **エンタープライズPMOニーズへの対応**  
   AsanaのエンタープライズシェアはPMO機能が評価されているが、
   PMO-AIはより専門的なPMO機能（リスクマトリクス・WBS・EVM等）で差別化可能。
