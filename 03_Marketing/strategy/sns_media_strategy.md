**作成日:** 2026年4月7日

---

## 1. 戦略概要

### 目的
AI PMOちゃんを「日本のPM・マネージャー層が最初に思い浮かべるAI PMOツール」としてポジショニングする。SNSを通じて以下の3つを同時に達成する。

| 目的 | 内容 |
|------|------|
| **認知獲得** | ターゲット層への製品・キャラクターの認知拡大 |
| **信頼構築** | PMO課題への知見・実績を通じたブランド権威性の確立 |
| **リード獲得** | SNS→LP→トライアル申込の転換フローの設計 |

### AI PMOちゃんのSNS戦略上の強み
AI PMOちゃんは**キャラクターを持つB2Bプロダクト**という希少な存在。競合（Asana・Backlog・ClickUp）はすべてロゴ・機能中心の無機質なSNS運用をしており、**親しみやすいキャラクターによる感情的つながり**は最大の差別化要素となる。

## 3. 媒体選定と優先度

### 優先度マップ

```
優先度A（コア）   ┌─────────────────────────────────────────┐
                  │  X（Twitter）  │  note  │  Instagram  │
                  └─────────────────────────────────────────┘

優先度B（増幅）   ┌─────────────────────────────────────────┐
                  │  YouTube  │  LinkedIn  │ 業界メディア  │
                  └─────────────────────────────────────────┘

優先度C（補完）   ┌───────────────────────────────┐
                  │  Facebook  │  Eight（Sansan）  │
                  └───────────────────────────────┘
```

### 優先度の根拠

| 媒体 | 優先度 | 選定理由 |
|------|--------|---------|
| **X** | A | 日本のPM・IT層の情報感度が最も高い。リアルタイム性とバイラル性でブランド認知を最速で広げられる |
| **note** | A | 購買検討層が深読みする媒体。SEO効果もあり、「PMO」「プロジェクト管理 AI」でロングテール流入を狙える |
| **Instagram** | A | AIPMOちゃんのキャラクター・ビジュアルを武器にできる唯一の媒体。競合が手薄でブルーオーシャン。サブターゲット（メンバー層）への認知にも有効 |
| **YouTube** | B | 購買検討期の「デモを見たい」「比較したい」ニーズに応える。意思決定の最終段階を後押しする |
| **LinkedIn** | B | 外資・コンサル系PMO・グローバル志向のIT部門長への直接リーチ。パートナー開拓にも有効 |
| **業界メディア** | B | 決裁者が信頼する第三者権威。プレスリリース・寄稿で「本物感」を担保する |
| **Facebook** | C | 40代以上の地方中小企業経営者層への補完。X/noteのRSS連携で最小工数で運用 |
| **Eight** | C | 展示会・ウェビナー後のフォローアップ専用。常時運用不要 |

### 4-1. X（旧Twitter）

**役割:** 認知獲得の最前線。問題提起・共感を通じてターゲット層のフォロワーを獲得する

| 項目 | 内容 |
|------|------|
| **主ターゲット** | PM・マネージャー・エンジニア（28〜40歳） |
| **投稿頻度** | 毎日1〜2投稿 |
| **最適投稿時間** | 平日 8:00-9:00（通勤）/ 12:00-13:00（昼休み）/ 21:00-22:00（帰宅後） |
| **KPI** | フォロワー数・インプレッション・引用RT数・プロフィールクリック率 |
| **数値目標** | Phase1: フォロワー500 / Phase2: フォロワー2,000 / Phase3: フォロワー5,000 |

#### コンテンツフォーマット（比率）

| フォーマット | 比率 | 内容 |
|-------------|------|------|
| **PMOあるある（共感投稿）** | 40% | 「〇〇あるある → AI PMOちゃんが解決」の型。問題提起→解決の2部構成 |
| **知識・Tip投稿** | 25% | 「PM必読」「プロジェクト管理の鉄則」等。リツイートされやすい教育コンテンツ |
| **キャラクター投稿** | 20% | AI PMOちゃんの一言・セリフ形式。「今日も全力でサポートするにゃん✨」等 |
| **時事連携投稿** | 10% | AI・DX・働き方改革のニュースへの反応。トレンドに乗る |
| **機能紹介・お知らせ** | 5% | 新機能・アップデート情報 |

#### トーン・文体ガイドライン
- **問題提起型:** 断定的・リズム感のある箇条書き。「〜だ。」で終わる文体
- **キャラクター型:** にゃん語尾・絵文字・ポップな口調（「できるよ〜！✨🐱」）
- **知識型:** 簡潔で実用的。「〇〇する3つの理由」「〜の落とし穴」

#### スレッド投稿テンプレート
```
【1/5】PMOあるある。
プロジェクトが炎上する「2週間前」、必ず予兆がある。
でも誰も言わない。数字にも出ていない。

【2/5】なぜか？
…（問題の深掘り）

【3/5】AI PMOちゃんなら…
…（解決策）

【4/5】実際のユーザーの声
…（社会的証明）

【5/5】今すぐ試してみる→
（LP URL） #PMO #プロジェクト管理 #AI活用
```

| シリーズ名 | 頻度 | 内容 |

| **AI PMO実践ガイド** | 月1本 | 「PMO自動化の始め方」等の実用ハウツー。SEO流入を狙う |

## 6. キャラクター活用ガイドライン

AIPMOちゃんのキャラクターは最大の差別化資産。媒体別に活用方法を変え、一貫したキャラクターブランドを構築する。

### キャラクター基本設定（SNS向け）
- **名前:** AI PMOちゃん（あいぴーえむおーちゃん）
- **語尾:** 「にゃん」「だよ〜」「できるよ！」
- **絵文字:** ✨🐱📋⚡🎯 を基本セット
- **性格:** 明るい・頼れる・でもかわいい・プロ意識高め
