# AI販売組織 — ルートコンテキスト

## プロダクト概要
**製品名:** PMO-AI（仮称）
**概要:** AIがPMO（プロジェクトマネジメントオフィス）業務を自動化・支援するSaaSプロダクト。
プロジェクト計画、タスク管理、進捗レポート、リスク管理、ステークホルダーコミュニケーションをAIが担う。

## 組織構造
```
CEO担当              ← 全体統括・意思決定
  ├── リサーチ担当    ← 競合分析・市場調査
  ├── マーケティング担当 ← 戦略立案・ブランディング・コンテンツ
  ├── セールス担当    ← 商談・クロージング・顧客対応
  ├── PM担当         ← タスク・進捗・スケジュール管理
  └── クリエイティブ制作担当 ← ビジュアル・素材・キャラクター世界観
```

## AI PMOちゃん プロダクトコンテキスト

**AI PMOちゃん**は、SlackなどのビジネスチャットツールにネイティブBOTとして常駐し、対話ベースでPMO（プロジェクトマネジメントオフィス）業務を自動化・支援するAIエージェントである。ユーザーは専用UIやダッシュボードを開くことなく、日常会話の延長でタスク管理・プロジェクト進捗確認・日程調整などの業務を完結できる。

### キャラクター・コミュニケーション特性
- **トーン:** 明るく・ポップ・フレンドリー（例：「にゃん！できるよ〜！✨🐱」）
- **催促スタイル:** 強制感ゼロ。人間から言われるより心理的抵抗が圧倒的に低い
- **キャラクター性:** 親しみやすいAIキャラ。チームの"同僚"的ポジション
- **対話スタイル:** 自然言語で話しかけるだけ（「タスク見せて」「進捗どう？」）

### 現在できること（実装済み機能）
- タスクの一覧表示（今日・期限切れ・担当者別・プロジェクト別）
- タスクのステータス更新
- プロジェクトの進捗確認・一覧表示
- チームへの励まし・雑談対応
- **Notion**との連携による読み取り＆ステータス更新

### 近日実装予定機能
タスクの新規追加、会議案内・リマインド設定、メンバーへのタスク自動割り当て・通知、週次レポート自動生成

### ターゲット顧客像
- **企業規模:** Slackを使う10〜300名規模のスタートアップ〜中堅企業
- **職種:** PM・PdM・プロジェクトリーダー・チームマネージャー
- **痛点:** タスク管理ツールを入れたが浸透しない・進捗把握が大変
- **ツール環境:** Slack + Notion を利用中の組織

### 競合との差別化ポイント
| 比較軸 | 既存ツール（Asana, Notionなど） | AI PMOちゃん |
|--------|-------------------------------|--------------|
| 操作方法 | ダッシュボード操作 | Slackで話しかけるだけ |
| 催促方法 | 通知・アラート | 対話形式・キャラ催促 |
| 心理的負担 | 高い（ツールを開く必要あり） | 低い（会話の延長） |
| ミス | ヒューマンエラーあり | AI管理で漏れなし |
| 導入コスト | ツール習熟が必要 | 自然言語のみでOK |

## 連携ルール
- 各担当は自分のフォルダ内の `CLAUDE.md` に従って行動する
- 他部門への依頼は `05_PM/tasks/` にタスクとして起票する
- CEO担当はすべての部門のアウトプットをレビューし最終承認する
- 週次レポートは `05_PM/reports/` に集約する
